Cross-Block Text Selection Test

Test Instructions

Please test the following scenarios in the journal editor at http://localhost:3001:

  1. Create multiple blocks with text content
  2. Test each scenario below
  3. Mark each test as PASS or FAIL based on the results

Test 1: Top-to-Bottom Selection PENDING

Steps:

  1. Click and drag from the beginning of an upper block to the end of a lower block
  2. Verify that text across multiple blocks is selected

Expected: Text selection should work smoothly from top to bottom

Test 2: Bottom-to-Top Selection PENDING

Steps:

  1. Click and drag from the end of a lower block to the beginning of an upper block
  2. Verify that text across multiple blocks is selected

Expected: Text selection should work smoothly from bottom to top

Test 3: Cross-Block Deletion with Delete Key PENDING

Steps:

  1. Select text across multiple blocks (either direction)
  2. Press the Delete key
  3. Verify that selected text is deleted and cursor is positioned correctly

Expected: Selected text should be deleted, blocks should merge if needed, cursor positioned correctly

Test 4: Cross-Block Deletion with Backspace Key PENDING

Steps:

  1. Select text across multiple blocks (either direction)
  2. Press the Backspace key
  3. Verify that selected text is deleted and cursor is positioned correctly

Expected: Selected text should be deleted, blocks should merge if needed, cursor positioned correctly

Test 5: Cross-Block Text Replacement PENDING

Steps:

  1. Select text across multiple blocks (either direction)
  2. Type new text (e.g., "REPLACED")
  3. Verify that selected text is replaced with the new text

Expected: Selected text should be replaced with new text, cursor positioned after new text

Test 6: Mixed Direction Operations PENDING

Steps:

  1. Test both top-to-bottom and bottom-to-top selections
  2. Perform deletion and replacement operations on both types
  3. Verify consistent behavior regardless of selection direction

Expected: All operations should work consistently regardless of selection direction

Test 7: Typography Font Awareness PENDING

Steps:

  1. Create blocks with different typography styles (if available)
  2. Test cross-block selection between blocks with different fonts
  3. Verify selection accuracy and cursor positioning
  4. Test deletion and replacement with mixed typography

Expected: Selection should work accurately regardless of font differences

Test 8: Font Loading States PENDING

Steps:

  1. Refresh the page and immediately try cross-block selection
  2. Test selection while fonts are still loading
  3. Verify that selection works after fonts have loaded

Expected: Selection should work correctly both during and after font loading
